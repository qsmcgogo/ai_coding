# 工程题 1：俄罗斯方块

## 题目描述

请在已有的项目框架中，实现俄罗斯方块的核心游戏逻辑。

你需要实现 `js/logic.js` 中的所有函数，使游戏具备以下功能：

1. **方块下落**：方块从顶部生成，按固定间隔自动下落
2. **消除**：当某一行被方块填满时，该行被消除，上方方块下移
3. **计分**：消除行数越多，得分越高；同时消除多行有额外加分

## 你需要实现的接口

在 `js/logic.js` 中实现以下函数：

| 函数 | 说明 |
|------|------|
| `createBoard(rows, cols)` | 创建空棋盘（二维数组，0 表示空） |
| `getPieceShape(type)` | 返回指定类型方块的形状矩阵 |
| `checkCollision(board, shape, row, col)` | 检测方块在指定位置是否碰撞 |
| `placePiece(board, shape, row, col, colorId)` | 将方块固定到棋盘上 |
| `clearLines(board)` | 消除满行，返回新棋盘和消除行数 |
| `calculateScore(linesCleared, level)` | 根据消除行数和等级计算得分 |
| `rotateShape(shape)` | 顺时针旋转方块 90 度 |

详细的接口说明请查看 `js/logic.js` 中的注释。

## 操作说明

实现完成后，可通过 **预览运行** 按钮试玩游戏：

- 方向键 ←→：左右移动
- 方向键 ↑：旋转
- 方向键 ↓：加速下落
- 空格键：直接落到底部
- P 键：暂停/继续
